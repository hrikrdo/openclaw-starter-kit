# TOOLS.md - Notas Locales de Herramientas

Los skills definen *cómo* funcionan las herramientas. Este archivo es para *tus* específicos — lo único de tu setup.

## ⚠️ Instrucción Importante

**SIEMPRE verifica credenciales/accesos ANTES de decir que no tienes acceso a algo:**

1. Revisar `~/.openclaw/secrets/` para credenciales
2. Revisar `mcporter list` para servidores MCP
3. Revisar skills disponibles en `~/clawdbot/skills/`
4. Solo después de verificar, confirmar si tienes o no acceso
